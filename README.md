# The Room Finder

## Description

Room Finder is a frontend web application that helps students find PGs and rental rooms easily.
It provides options for students to search rooms and for owners to add room listings.

## Features

* User Login Interface
* Search rooms by budget and category
* Boys / Girls accommodation selection
* Single / Shared room options
* Owner can add PG details

## Tech Stack

* HTML
* CSS
* JavaScript

## Setup Instructions

1. Clone the repository:
   https://github.com/knyyshka/The-Room-Finder

2. Open the project in VS Code

3. Open `index.html` in browser or Live Server

## Pages

* Login Page
* User Dashboard
* Room Listings
* Owner Add Room Page
